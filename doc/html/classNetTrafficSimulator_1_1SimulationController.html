void NetTrafficSimulator.SimulationController.createLinks ( )
inlineprivate

For each pair of nodes where connection is marked, create a new Link instance and register the link to each node. New links are stored in LinkedList<Link> links initialized here (as their count is unknown)

As link capacities are not supported by NetworkModel yet, all are set for 0 at this point.

Test is necessary to verify links are created properly according to the model

Exceptions
InvalidOperationExceptionif any node is of unidentified type, the network model is null, the nodes array is null or length of the nodes array don't match node count in the network model
